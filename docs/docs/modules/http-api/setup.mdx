# http-api

### Installation

`minimalist-java` is available on GitHub Packages. To use it, add the following dependency to
your `pom.xml` file:

<PomDependency artifactId="http-api"/>

You can check how to authenticate with GitHub Packages [here](../../installation/from-github).
